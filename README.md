# ID Document Forgery Detection

##  Project Overview
This project detects whether an ID document is genuine or suspicious using image processing and OCR.

## Technologies Used
- Python
- Flask
- OpenCV
- Tesseract OCR

##  Features
- Blur detection
- Edge detection
- Text extraction (OCR)
- Risk score calculation
- Final status (Genuine / Suspicious)

##  How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Install Tesseract OCR

3. Run app:
python app.py

4. Open:
http://127.0.0.1:5000

##  Output
- Risk Score
- Confidence %
- Final Status
